# MVP de Machine Learning para Volatilidade do Bitcoin

Este repositório contém os arquivos utilizados em um MVP de Machine Learning para classificar períodos futuros de alta volatilidade do Bitcoin.

O projeto foi desenvolvido como parte de uma atividade acadêmica de Machine Learning e Analytics. O objetivo é avaliar se dados históricos públicos de mercado do Bitcoin podem ajudar a identificar períodos em que os 30 dias seguintes tendem a apresentar alta volatilidade.

## Arquivos do repositório

| Arquivo | Descrição |
|---|---|
| `btc_binance_ml_dataset.csv` | Dataset final processado, usado no MVP de Machine Learning |
| `gerar_dataset_binance_btc_ml.ipynb` | Notebook opcional que documenta, por transparência, como os dados foram baixados da Binance Public Data e transformados no CSV final; não é necessário para executar o MVP principal |
| `MVP_ML_Analytics_20261_FilipeFCalvario.ipynb` | Notebook principal do MVP, com definição do problema, análise exploratória, modelagem, avaliação e conclusão |
