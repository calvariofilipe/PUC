Este repositório contém o dataset processado utilizado em um MVP de Machine Learning para classificar períodos futuros de alta volatilidade do Bitcoin.

O projeto foi desenvolvido como parte de uma atividade acadêmica de Machine Learning e Analytics. O objetivo é avaliar se dados históricos de mercado do Bitcoin podem ajudar a identificar períodos em que os 30 dias seguintes tendem a apresentar alta volatilidade.
